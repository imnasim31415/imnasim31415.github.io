### [Red Hat Certified System Administrator (RHCSA)](https://www.credly.com/badges/bc987621-22d8-4484-9f16-7cdde481f4ae)

**Issuer:** Red Hat &nbsp;|&nbsp; **Issued:** Dec 2025 &nbsp;|&nbsp; **Expires:** Dec 2028 &nbsp;|&nbsp; **Credential ID:** 250-195-955

Industry-recognized credential validating hands-on Linux system administration skills in enterprise environments. The exam is a **100% practical, command-line-only performance test** ‚Äî no multiple choice.

**Topics covered:**
- Essential Linux tools ‚Äî file management, archiving, text processing, shell scripting
- User and group administration, file permissions, ACLs, `sudo` policies
- Storage management ‚Äî partitions, LVM, `stratis`, swap, NFS/autofs mounts
- Networking ‚Äî static IPs, hostname resolution, firewall (`firewalld`)
- System services ‚Äî `systemd` units, boot targets, scheduling with `cron` and `at`
- Package management with `dnf`, managing software repositories
- SELinux ‚Äî enforcing mode, contexts, troubleshooting policy violations
- Containers ‚Äî basic deployment with `podman`, persistent storage, port mapping

### [CS50 ‚Äî Introduction to Databases with SQL](https://certificates.cs50.io/279c4c4b-8e14-4745-9b9e-29f4c5bcc6a0.pdf?size=letter)

**Issuer:** CS50, Harvard University &nbsp;|&nbsp; **Issued:** Dec 2024

Harvard's intensive introduction to database design and SQL, covering relational theory through real-world problem sets using SQLite, MySQL, and PostgreSQL.

**Topics covered:**
- Relational model ‚Äî tables, primary/foreign keys, normalization, schema design
- Core SQL ‚Äî `SELECT`, `INSERT`, `UPDATE`, `DELETE`, filtering, aggregation, grouping
- Joins ‚Äî `INNER`, `LEFT`, `RIGHT`, `FULL OUTER`, self-joins, cross-joins
- Subqueries, CTEs (`WITH`), and nested queries
- Indexes ‚Äî B-tree internals, query optimization, `EXPLAIN` plans
- Views and materialized views
- Transactions ‚Äî ACID properties, `BEGIN`, `COMMIT`, `ROLLBACK`, isolation levels
- Constraints and data integrity ‚Äî `NOT NULL`, `UNIQUE`, `CHECK`, `FOREIGN KEY`

### [LFS101: Introduction to Linux](https://www.credly.com/badges/2a3487f5-1cb8-47af-a4d9-356e263055a5/linked_in_profile)

**Issuer:** The Linux Foundation &nbsp;|&nbsp; **Issued:** Mar 2024

Foundational course by The Linux Foundation covering core Linux concepts and command-line proficiency across major distributions (Debian, Red Hat, openSUSE families).

**Topics covered:**
- Linux history, distributions, and the open-source ecosystem
- Graphical desktop environments (GNOME) and the X Window System
- Filesystem hierarchy standard (FHS) ‚Äî navigating and managing directories
- Command-line essentials ‚Äî `bash`, pipes, redirection, environment variables
- File operations ‚Äî permissions (`chmod`, `chown`), links, archiving (`tar`, `gzip`)
- Text editors ‚Äî `vi`/`vim`, `nano`, `gedit`
- Process management ‚Äî `ps`, `top`, signals, background jobs
- Network fundamentals ‚Äî `ip`, `ping`, `ssh`, `curl`, `wget`
- Package management across distributions (`apt`, `dnf`, `zypper`)
- System services, logging (`journalctl`), and basic security practices
